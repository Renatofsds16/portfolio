# 🚀 Portfolio Fullstack - Django + Next.js

Este é um projeto de portfólio pessoal desenvolvido com **Django (backend)** e **Next.js (frontend)**, com foco em performance, organização e facilidade de atualização via painel administrativo.

---

## 📸 Preview

> O conteúdo do portfólio é dinâmico e gerenciado totalmente pelo Django Admin.

---

## 🧠 Sobre o Projeto

Este sistema foi criado para funcionar como um portfólio dinâmico, onde todas as informações podem ser atualizadas diretamente pelo painel administrativo do Django, sem necessidade de mexer no código do frontend.

---

## ⚙️ Tecnologias Utilizadas

### Backend
- Django
- Django REST Framework
- SQLite
- Pillow (para imagens)

### Frontend
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
